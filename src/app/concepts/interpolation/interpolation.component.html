About {{ titles.titleInterploation }}

Main Points

  • Use to display dynamic data in HTML page

    Dynamic data will be coming from TS file, where we create the variable either for String, boolen value or numerical values

  • Types of Interpolation

    1. Simple Interpolation
                      my.component.html
                Simple Interpolation
                { { myTitle }}
              
                my.component.ts
                myTitle: string = "My Tiltle Goes Here"
    2. Interpolation using Objects
                      
                my.component.html
                Object Interpolation
                  { { titles.firstTitle }}
                  { { titles.subTitle }}
                
                  my.component.ts
                  titles={
                    firstTitle: "My First Title"
                    subTitle: "My Sub-Title"
                    }
                  myTitle: string = "My Tiltle Goes Here"
    3. Interpolation using Function
                      
                my.component.html
                Function Interpolation
                { { getValue() }}
                Output
                Interpolation Value coming from { { My Title }} via function

                my.component.ts
                getValue(){
                  return "My Title"
                }
    4. Interpolation: check value is correct or not using Boolean
                    my.component.html
              Boolean Interpolation
              { { title=="myTitle" }}
              Output
              Interpolation Value coming from { { My Title }} via function

              my.component.ts
              title = "Interpolation"
    5. Interpolation: in css class
                  my.component.html
            Interpolation as CSS Class
            

            Output
            <p class="{ {title}}">Value used in class</p>

            my.component.ts
            title = "myTitle"
